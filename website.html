About the Website

Accessibility

  1. 3.2.3 Consistent Navigation (02/22/2020): The navigation is the same across all pages of the website with content.
  2. 2.4.6 Headings and Labels (02/22/2020): Each part of the website has headings and labels to let the user know what they are looking at.
  3. 1.4.1 Color (03/03/2020): Color is not the only meaningful way of conveying information of the webpage.
  4. 2.4.4 Link Context (03/03/2020): When creating the website, I made sure that the purpose of each link can be inferred or determined by the text.
  5. 2.2.3 No Timing (03/10/2020): This website does not implement any features which use timing. The readers can take as much or as little time as they want on each page.
  6. 2.4.2 Page Titled (03/10/2020): Each page of the website has a title that is meant to let the user know what information is written on that page about the research project.
  7. 2.4.8 Location (04/03/2020): The user is able to see where they are located in relation to other pages by looking at the navigation bar. The page that the user is on has font in a different color.
  8. 1.3.3 Sensory Characteristics (04/03/2020): Understanding or operating the webpage content does not rely solely on sensory characteristics.
  9. 1.4.7 Low or No Background Audio (04/11/2020): There is no background audio included in this web page.
  10. 1.4.12 Text Spacing (04/11/2020): There is no loss of content due to text spacing.
  11. 2.1.1 Keyboard Accessible (04/19/2020): All functionality of the website is accessible by use of keyboard.
  12. 2.2.4 Interruptions (04/19/2020) : There are no interruptions or pop ups on the website.
  13. 1.4.10 Reflow (04/26/2020) : No loss of functionality while scrolling.
  14. 2.4.5 Info and Relationships (04/26/2020) : Relationships are conveyed through presentation
